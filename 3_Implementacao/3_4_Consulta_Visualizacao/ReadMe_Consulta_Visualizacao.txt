ReadMe - Consulta e Visualização

Este diretório corresponde à fase de consulta e visualização do projeto StreamFlix, onde os dados armazenados no modelo dimensional são analisados através de dashboards interativas desenvolvidas em Power BI.

➡ Estrutura da Pasta

    "CuboStreamFlix/" – Projeto do Visual Studio 2022 utilizado para a criação do cubo OLAP a partir do modelo dimensional em SQL Server;

    "StreamFlix.pbix" – Ficheiro do Power BI com as dashboards desenvolvidas;

    "Configurar_OLAP.pdf" – Documento que explica passo a passo como configurar o cubo OLAP no Visual Studio e prepará-lo para ser utilizado no Power BI;

    "Explicacao_Dashboards_OLAP.pdf" – Documento descritivo das duas dashboards construídas, detalhando os componentes visuais e funcionalidades interativas utilizadas (ex: slice and dice, drill through, filtros).
